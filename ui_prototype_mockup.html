💡 交互亮点提示：

  • 家庭资料管理：在日历页面左上角可点击切换不同的家庭组合，每次进入会默认展示您最后一次选中的家庭。
  • 日历视图状态：日历格子中，有排餐计划的日期下方会自动显示指示红点，直观且节约空间。
  • 大模型一键排餐：在日历下方的“今日菜谱”卡片，点击“一键智能排餐”可体验智能一键生成本周的排餐（伴有AI高光加载动画）。
  • 单日编辑与推荐：点击任意一个日历格子，系统将加载当天的详情。点击“编辑今日菜谱”可进入编辑单日页面，拥有读写权限的人可点击“AI一键推荐”获取今日配菜方案。
  • 共享视图模式限制：切换成“做饭阿姨”，进入页面将仅可浏览菜单，相关AI及编辑保存操作不可见，并展示温馨的浏览提示。
  • 系统设置锁定保护：切换为“家庭成员”或“做饭阿姨”，系统设置Tab会被锁定保护，仅创建者可见，确保 API Key 等系统数据安全。
